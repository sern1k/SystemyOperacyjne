Systemy Operacyjne, zadanie 2 , Krakow, 22.03.2022, Pola Madej, Informatyka Stosowana 1 rok, numer albumu 1181441



Zawartosc katalogu:
	1. pliki .c (main.c, wypisywanie.c) - pliki z kodem zrodlowym w jezyku C
	2. plik odp.txt - zawiera odpowiedz na zadane pytanie
	3. plik README - plik, ktory zawiera podstawowe informacje o programie oraz zawartosci katalogu
	4. plik Makefile



Program sluzy do zapoznania sie z dzialaniem komendy execlp i zauwazeniem roznicy pomiedzy poprzednim zadaniem a tym.



Komendy przydatne do korzystania z programow:
	- make run - uruchamia program
	- make clean - czysci katalog
	- make tar - kompresuje katalog
